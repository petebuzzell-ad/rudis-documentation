RUDIS Technical Documentation

Comprehensive technical documentation for the RUDIS Shopify Plus eCommerce platform. This documentation provides evidence-based technical information, performance baselines, accessibility compliance guides, and operational procedures.

Platform Documentation

  • Platform Documentation
    Comprehensive platform documentation hub with page types, customizer sections, image specifications, and quick reference guides. Includes links to Technical User Guide (for developers) and Business User Guide (for content managers).
  • Business User Guide
    Step-by-step guides for content managers and marketing teams. Includes team store setup, product template management, collection configuration, and troubleshooting guides.
  • Technical User Guide
    Complete technical documentation for developers including template assignment system, customizer sections, image specifications, and custom features documentation.
  • Data Guide
    Comprehensive reference for all metafields, metaobjects, and custom data structures used in the RUDIS platform. Includes field types, usage patterns, and code references.
  • Team Store / B2B System
    Complete technical architecture documentation for the Team Store B2B system including Celigo integration, NetSuite integration, custom cart logic, and approval workflow.
  • Performance Documentation
    Core Web Vitals baseline, performance optimization roadmap, and monitoring procedures. Includes field data analysis and synthetic testing results.
  • Accessibility Documentation
    WCAG 2.2 AA compliance status, accessibility violations, and remediation guide with prioritized action items.

Requests & Analysis

  • RUDIS JIRA Insights
    Development budget allocation analysis, issue categorization, and strategic area breakdowns based on JIRA data from 2025.
