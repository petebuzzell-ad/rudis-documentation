Image Specifications

General Guidelines

  • Formats: JPG (photographs), PNG (transparency), WebP (optimized)
  • File Sizes: Desktop <500KB, Mobile <200KB
  • Optimization: Always compress images before upload

Standard Image Sizes

Use Case Dimensions Aspect Ratio Format
Banner Images (Desktop) 1920px × 1080px 16:9 JPG, PNG
Banner Images (Mobile) 750px × 422px 16:9 JPG, PNG
Product Images (Square) 1200px × 1200px 1:1 JPG, PNG
Product Images (Portrait) 1200px × 1500px 4:5 JPG, PNG
Column/Card Images (Square) 800px × 800px 1:1 JPG, PNG
Column/Card Images (Portrait) 800px × 1200px 4:5 JPG, PNG

Theme Structure

File Organization

  • Sections: 124 files (main templates and reusable components)
  • Snippets: 183 files (reusable components, icons, integrations)
  • Templates: 413 files (403 JSON templates, 10 Liquid templates)
  • Assets: 182 files (126 CSS, 43 JS, 5 fonts, 8 other)

Directory Structure

theme_export/
├── assets/          # CSS, JS, fonts, images (182 files)
├── blocks/          # AI-generated blocks (10 files)
├── config/          # Theme settings and configuration
├── layout/          # Theme layouts (4 files)
├── locales/         # Translation files (multi-language support)
├── sections/        # Theme sections (124 files)
├── snippets/        # Reusable components (183 files)
└── templates/       # Page templates (413 files)
